# 替代梯度超参数搜索结果报告

## 实验概述
本实验旨在寻找在 CIFAR-10 数据集上训练脉冲神经网络 (SNN) 时，不同替代梯度 (Surrogate Gradient) 函数的最佳 `alpha` (文献中常称为 `beta`) 参数。我们使用了 **Optuna** 框架进行自动化的超参数搜索。

## 搜索空间配置
不同的替代梯度函数对陡峭度参数的敏感度和取值范围要求不同。分别定制的搜索空间：

| 替代梯度函数 | 搜索范围 | 缩放尺度 | 描述 |
| :--- | :--- | :--- | :--- |
| **SuperSpike** | `1.0` - `200.0` | **对数 (Log)** | 控制梯度近似的宽度。SuperSpike 通常需要较大的值来有效近似 Dirac delta 函数，因此采用对数尺度和较大的范围。 |
| **Sigmoid** | `0.1` - `20.0` | 线性 (Linear) | 控制 Sigmoid 函数的陡峭程度。这是一个覆盖平滑到尖锐过渡的标准范围。 |
| **Esser** | `0.1` - `10.0` | 线性 (Linear) | 控制线性截断的斜率。 |

## 优化结果

> **注**: 以下结果基于 Optuna 搜索得出。

| 替代梯度函数 | 最佳 Alpha | 最佳测试准确率 (%) |
| :--- | :--- | :--- |
| **SuperSpike** | `6.21` | **73.07%** |
| **Sigmoid** | `5.55` | **73.47%** |
| **Esser** | `0.91` | **72.21%** |

## 实验观察

1.  **Sigmoid 表现最佳**: 在本次实验中，Sigmoid 替代梯度取得了最高的测试准确率 (73.47%)，其最优 alpha 约为 5.55。
2.  **SuperSpike**: 尽管搜索空间很大 (1.0 - 200.0)，但最优值落在了较小的范围 (~6.21)，这表明过大的 alpha 可能导致梯度过窄，不利于反向传播。
3.  **Esser**: Esser 函数偏好较小的 alpha 值 (~0.91)，这与其作为线性截断函数的特性相符，较小的斜率提供了更宽的梯度回传窗口。
